## ⚙ 설치

```
# clone the project
$ git clone https://github.com/Space-Belt/job-hang-a-sa

# install modules
$ cd job-hang-a-sa
$ npm install || yarn install

# start
$ npm run dev || yarn dev

⠀
⠀  You can now view this project in the browser.
⠀  http://localhost:3000/
⠀
```
